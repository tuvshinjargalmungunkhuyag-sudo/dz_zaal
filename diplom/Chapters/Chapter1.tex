\section*{\centering \Large УДИРТГАЛ}

\subsection*{Сэдэв сонгосон үндэслэл}

Мэдээллийн технологийн хурдацтай хөгжил нь орчин үеийн иргэдийн өдөр тутмын амьдралыг ихээхэн өөрчилж байна. Цахим захиалга, онлайн үйлчилгээ нь дэлхийн өнцөг булан бүрт нэвтэрч, хүмүүсийн цаг хугацааг хэмнэж, үйлчилгээний чанарыг дээшлүүлж байна. Гэвч Монгол улсын орон нутгийн хотуудад, тухайлбал Өмнөговь аймгийн Даланзадгад хотод, уламжлалт үйлчилгээний хэлбэр давамгайлсаар байна.

Даланзадгад хотод өнөөдрийн байдлаар хэд хэдэн спорт заал, тамирын цогцолбор үйл ажиллагаа явуулж байгаа боловч иргэд цаг захиалахдаа дараах бэрхшээлтэй тулгардаг:

\begin{itemize}
    \item Утсаар залгаж цаг захиалах үед шугам завгүй байх, буруу ойлголцол үүсэх;
    \item Биечлэн очиж захиалахад цаг, зардал зарцуулагдах;
    \item Боломжтой цагийн мэдээлэл тодорхойгүй, ил тод бус байх;
    \item Захиалгаа цуцлах, өөрчлөх боломж хязгаарлагдмал байх;
    \item Захиалгын баталгаажуулалт, сануулга авах боломжгүй байх.
\end{itemize}

Эдгээр асуудлыг шийдвэрлэхийн тулд орчин үеийн мобайл технологийг ашиглан цаг захиалгын систем хөгжүүлэх нь чухал шаардлага болж байна. \keyword{Flutter} фреймворк нь нэг удаа бичиж Android болон iOS платформд нэгэн зэрэг гаргах боломж олгодог тул хөгжүүлэлтийн зардал, хугацааг мэдэгдэхүйц бууруулдаг. \keyword{Firebase} бэкэнд үйлчилгээ нь бодит цагийн өгөгдөл солилцоо, найдвартай нэвтрэлт, тасралтгүй ажиллагааг хангадаг. Харин \keyword{Node.js/Express} бэкэнд, \keyword{Docker} контейнержуулалт, \keyword{AWS} дэд бүтэц нь системийг үүлэн орчинд байршуулж, тасралтгүй хүргэх боломжийг олгодог.

%-------------------------------------------------------------------------------

\subsection*{Зорилго}

\textbf{Зорилго:} Өмнөговь аймгийн Даланзадгад хотын спорт заалнуудын цаг захиалгыг цахимжуулах, хэрэглэгчдэд хялбар, хурдан, ил тод байдлаар захиалга хийх боломж олгох \keyword{Flutter} мобайл аппликейшн болон \keyword{Node.js/Express} бэкэнд бүхий 3 давхаргат системийг \keyword{үүлэн орчинд} (AWS) хөгжүүлж, байршуулах.

%-------------------------------------------------------------------------------

\subsection*{Зорилт}

Уг зорилгод хүрэхийн тулд дараах зорилтуудыг дэвшүүлж байна. Үүнд:

\begin{enumerate}
    \item Даланзадгад хотын спорт заалнуудын цаг захиалгын одоогийн нөхцөл байдал, иргэдийн хэрэгцээг судлах;
    \item Цаг захиалгын мобайл апп-ын шаардлагын дүн шинжилгээ хийх, системийн загварчлал боловсруулах;
    \item Flutter болон Firebase технологийг ашиглан 3 давхаргат системийн архитектур боловсруулах;
    \item Хэрэглэгчийн UI/UX загвар гаргаж, мобайл аппликейшн хэрэгжүүлэх;
    \item Node.js/Express бэкэнд хөгжүүлж, Docker ашиглан AWS EC2-д байршуулах;
    \item GitHub Actions CI/CD хоолойгоор автоматжуулалт хэрэгжүүлэх;
    \item Системийн гүйцэтгэл, хэрэглэгчийн сэтгэл ханамжийг туршиж, үнэлэх.
\end{enumerate}

\subsection*{Судлагдсан байдал}

Спорт заал болон үйлчилгээний цаг захиалгын мобайл аппликейшн хөгжүүлэлтийн чиглэлд олон улсын болон дотоодын судлаачид тодорхой хэмжээний ажил хийсэн байна.

Олон улсын түвшинд \textbf{Mindbody} (АНУ), \textbf{Vagaro} (АНУ) зэрэг цаг захиалгын платформууд спорт болон фитнесийн салбарт өргөн хэрэглэгдэж байна. Эдгээр систем нь iOS болон Android-д ажилладаг хэдий ч дотоодын зах зээлд, ялангуяа орон нутгийн хэрэгцээнд тохиромжтой нутагшсан шийдэл биш юм.

Монголын хэмжээнд Амарсайханы Билгүүн \cite{thesis_mobile} мобайл аппликейшн хөгжүүлэлтийн аргачлалын асуудлыг судалсан бол Т.Амартүвшин, Б.Зоригтбаатар нар \cite{mobile_mongol} мобайл апп хөгжүүлэлтийн онолын үндсийг тодорхойлсон байна. Гэхдээ тухайлан \textbf{үүлэн орчинд суурилсан}, \textbf{3 давхаргат архитектур} бүхий спорт заалны цаг захиалгын тусгай системийн талаарх судалгаа орон нутгийн хэмжээнд хийгдээгүй байна.

Иймд энэхүү ажил нь орон нутгийн хэрэгцээнд нийцсэн, орчин үеийн технологийн бүрэн стек ашигласан цоо шинэ шийдлийг санал болгож байна.

\subsection*{Шинэлэг тал}

Энэхүү ажлын шинэлэг тал нь дараахид оршино:

\begin{enumerate}
    \item \textbf{3 давхаргат архитектур:} Flutter клиент --- Node.js/Express бэкэнд --- Firebase/AWS дэд бүтэц гэсэн тодорхой давхаргаар системийг зохион байгуулсан;
    \item \textbf{Үүлэн орчин:} Docker контейнержуулалт болон AWS EC2 дэд бүтцийг ашиглан тасралтгүй, хэмжээгээр уян хатан системийг бий болгосон;
    \item \textbf{CI/CD автоматжуулалт:} GitHub Actions ашиглан код шинэчлэгдэх бүрт автоматаар байршуулдаг хоолой тавьсан;
    \item \textbf{AI чатбот:} Anthropic Claude Haiku болон Groq Llama загваруудыг нөөцлөлтийн горимоор нэгтгэж, хэрэглэгчийг туслах ухаалаг системийг хэрэгжүүлсэн;
    \item \textbf{Орон нутгийн хэрэгцээ:} Даланзадгад хотын таван спорт заалд тохирсон байгууллагуудын тогтмол захиалга (\code{fixed\_bookings}) зэрэг тусгай функцийг хэрэгжүүлсэн.
\end{enumerate}

\subsection*{Судалгааны хамрах хүрээ}

\textbf{Хамрах хүрээ:} Энэхүү ажлын хүрээнд дараахи зүйлс орно:

\begin{itemize}
    \item Иргэдэд зориулсан мобайл апп (Android болон iOS);
    \item Firebase дээр суурилсан email/нууц үгийн нэвтрэлт (Nodemailer 6 оронтой код баталгаажуулалттай, rollback механизмтай), өгөгдлийн сан (4 collection);
    \item Node.js/Express REST API бэкэнд;
    \item Docker контейнержуулалт, AWS EC2/ECR байршуулалт;
    \item GitHub Actions CI/CD автоматжуулалт;
    \item AI чатботын үйлчилгээ (Claude Haiku, Groq Llama);
    \item Локал мэдэгдлийн систем (flutter\_local\_notifications).
\end{itemize}

Уг ажилд төлбөрийн систем нэгтгэх болон GPS байршлын нарийн функцүүд хамрагдахгүй болно.

\subsection*{Ач холбогдол}

\textbf{Практик ач холбогдол:} Даланзадгад хотын иргэдэд спорт заал захиалах үйл явцыг хялбарчилж, цаг хугацааг хэмнэнэ. Спорт заалны ажилтнуудад захиалгын менежментийг автоматжуулж, ажлын ачааллыг бууруулна.

\textbf{Шинжлэх ухааны ач холбогдол:} Flutter болон Firebase технологийг Node.js/Express REST API, Docker/AWS дэд бүтэц, GitHub Actions CI/CD автоматжуулалт, AI чатбот үйлчилгээтэй хосолсон орчин үеийн бүрэн стекийн туршлагыг Монголын орон нутгийн нөхцөлд хэрэглэж буй анхны бүтээлүүдийн нэг болно.

\subsection*{Ажлын бүтэц}

Энэхүү бакалаврын төгсөлтийн ажил дараахи бүтэцтэй:

\textbf{Удиртгал:} Судалгааны үндэслэл, зорилго, зорилт, судлагдсан байдал, шинэлэг тал, хамрах хүрээ, ач холбогдлыг тодорхойлсон.

\textbf{1-р бүлэг --- Онолын үндэс ба технологийн судалгаа:} Мобайл апп хөгжүүлэлтийн орчин үеийн арга хандлага, Flutter фреймворк, Firebase үйлчилгээ, Node.js/Express, Docker, AWS, GitHub Actions, AI технологиудын онолын үндсийг судалсан. Ижил төстэй системүүдтэй харьцуулсан.

\textbf{2-р бүлэг --- Системийн зохион байгуулалт ба хэрэгжүүлэлт:} Шаардлагын дүн шинжилгээ, системийн 3 давхаргат архитектур, UML диаграммууд, өгөгдлийн сангийн загвар, Flutter апп болон Node.js бэкэндийн хэрэгжүүлэлт, Docker/AWS байршуулалт, CI/CD автоматжуулалтыг тусгасан.

\textbf{3-р бүлэг --- Системийн туршилт ба үнэлгээ:} Функциональ туршилт, нэгтгэлийн туршилт, гүйцэтгэлийн хэмжилт, хэрэглэхэд хялбар байдлын үнэлгээний үр дүнг тусгасан.

\textbf{Дүгнэлт:} Ажлын үр дүн, цаашдын хөгжлийн чиглэлийг дүгнэсэн.
