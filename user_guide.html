Захиалга хийх алхам алхмаар

1

Апп-ыг нээж, бүртгүүлэх эсвэл нэвтрэх

Апп-ыг анхлан нээхэд нэвтрэлтийн дэлгэц гарч ирнэ. Шинэ хэрэглэгч бол "Бүртгүүлэх" товчийг дарж: нэр, email, нууц үгээ оруулна. Системийн сервер таны email рүү 6 оронтой баталгаажуулах код илгээнэ (Nodemailer/Gmail SMTP). Кодыг оруулж баталгаажуулсны дараа нэвтэрч болно.

💡 Аль хэдийн бүртгэлтэй бол email болон нууц үгээ оруулж "Нэвтрэх" дарна уу.
2

Нүүр хуудаснаас спорт заал сонгох

Нэвтэрсний дараа нүүр хуудас нь 4 спорт заалыг үнэлгээгээр эрэмбэлэн харуулна. Дээрх хайлтын мөрөнд "сагсан бөмбөг" эсвэл "говийн" гэж бичиж хайж болно. Шүүлтүүр (⚙️) товчоор үнэ эсвэл нээлттэй байдлаар шүүх боломжтой.

💡 Заалны картыг дарахад захиалгын дэлгэц доороос гулсан гарч ирнэ.
3

Талбайн төрөл сонгох

Захиалгын дэлгэцэд хоёр сонголт байна:

  • Бүтэн талбай — бүтэн заалны үнэ (жишээ: 15,000₮/цаг)
  • Хагас талбай — үнэний хагас (7,500₮/цаг), 2 дугуй хагасын дотор нэг талыг авна

💡 Хагас талбайд 2 бүлэг нэгэн зэрэг ажиллаж болно.
4

Өдөр сонгох (14 хоногийн хуваарь)

Хэвтээ скролл хийж өнөөдрөөс эхлэн 14 хоног хүртэл урьдчилан захиалах боломжтой. Өдөр бүр сонгохын зэрэг тухайн өдрийн цагийн хуваарь Firebase-с татаж ачаалагдана.

5

Цагийн үеийг сонгох (08:00–00:00)

3 баганатай торонд 16 цагийн үе харагдана. Цагийн үеүдийн утга:

Нээлттэй — захиалах боломжтой
Захиалагдсан — хаалттай
Гэрээт (байгуулла) — хаалттай
Таны сонгосон цаг

Зэрэгцэх цагуудыг дараалан дарж олон цаг сонгох боломжтой (жишээ: 12:00–14:00 = 2 цаг).

💡 Захиалгын доод баар дахь нийт үнэ, цагийн мэдээлэл автоматаар тооцогдоно.
6

"Захиалах" товч дарах

Цаг сонгосны дараа доод хэсэгт "Захиалах" товч идэвхтэй болно. Дарахад баталгаажуулах дэлгэц руу шилжинэ.

7

Мэдээлэл болон төлбөрийн хэлбэр оруулах

Баталгаажуулах дэлгэцэд:

  • Нэр — профайлаас автоматаар дүүргэгдэнэ (засах боломжтой)
  • Email — нэвтэрсэн хаяг (засах боломжгүй, аюулгүй)
  • Төлбөр — QPay / Карт / Бэлэн гэсэн 3 сонголт
  • Дэлгэцийн дээд хэсэгт захиалгын нэгтгэл (огноо, цаг, байршил, үнэ) харагдана

8

"Захиалга баталгаажуулах" дарах → Амжилттай!

Серверт илгээж, давхцал шалгасны дараа захиалга Cloud Firestore-д хадгалагдана. Амжилтын дэлгэц гарч ирэх бөгөөд:

  • DBK-XXXXX форматын давтагдашгүй захиалгын дугаар олгогдоно
  • Локал сануулга 1 цагийн өмнө тохируулагдана
  • "Миний захиалгууд" табаас захиалгаа харах боломжтой
  • Захиалгаас өмнө "Цуцлах" боломжтой

💡 Захиалгын цаг дууссаны дараа сервер автоматаар "Дууссан" төлөвт шилжүүлнэ (cron ажил, 15 минут тутам).
4

Апп-ын бүрэн функцүүд

🔐

Email баталгаажуулалт

Бүртгэлийн үед Gmail SMTP-ээр 6 оронтой OTP код илгээж баталгаажуулна. Хэрэв алдаа гарвал Firebase хэрэглэгчийг rollback хийнэ.

🔍

Хайлт ба шүүлтүүр

Заалны нэр, төрөл, байршлаар хайж, үнэ эсвэл үнэлгээгээр эрэмбэлэх, зөвхөн нээлттэй заалнуудыг шүүх боломжтой.

📅

14 Хоногийн Хуваарь

Өнөөдрөөс эхлэн 14 хоногийн урьдчилсан захиалга хийх боломжтой. Цаг бүр бодит цагийн мэдээлэлтэй.

⏰

Цагийн Сонголт

08:00–00:00 хооронд 16 цагийн үе. Зэрэгцэх цагуудыг дарааллан сонгож олон цагийн захиалга хийнэ.

🏟️

Бүтэн/Хагас Талбай

Нэг цагт бүтэн талбай эсвэл хоёр хагас талбай зэрэг захиалагдах боломжтой. Үнэ автоматаар тооцогдоно.

🟣

Гэрээт Цагуудын Харагдалт

Байгууллагуудын (засгийн газар, компани) долоо хоногийн тогтмол захиалгыг ягаан өнгөөр тэмдэглэж харуулна.

🔔

Локал Сануулга

Захиалга баталгаажсаны дараа тухайн цагийн 1 цагийн өмнө локал мэдэгдэл тохируулагдана. FCM сервер шаардагдахгүй.

❌

Захиалга Цуцлах

"Хүлээгдэж буй" захиалгыг цуцлах боломжтой. Хүсэлт серверт очиж Firestore-д "cancelled" төлөвт шинэчлэгдэнэ.

🤖

AI Туслагч Чатбот

Claude Haiku 4.5 (Firebase Cloud Functions) үндсэн горим, Groq Llama 3.1-8b нөөц горим болгон ажиллана.

📊

Профайл Статистик

Нийт болон хүлээгдэж буй захиалгын тоог профайл дэлгэцэд харуулна. Гишүүний мэдээлэл, email харагдана.

Системийн архитектур

3 давхаргат архитектур:

1. Клиент давхарга — Flutter мобайл апп (Android & iOS)
    ↓ HTTP/REST + Firebase SDK
2. Логикийн давхарга — Node.js/Express бэкэнд (AWS EC2, 13.112.91.27:3000)
    ↓ Firebase Admin SDK
3. Өгөгдлийн давхарга — Cloud Firestore (4 collection) + AWS ECR (Docker images)

Firestore өгөгдлийн сан (4 collection)

CollectionЗориулалтГол талбарууд
bookings Цаг захиалгуудын мэдээлэл venueId, userEmail, date, timeSlot, timeSlotEnd, courtType, price, status, code (DBK-XXXXX)
users Бүртгэлтэй хэрэглэгчид uid, name, email, emailVerified, createdAt
fixed_bookings Байгууллагуудын гэрээт цагууд venueId, organizationName, weekDays, startHour, endHour
email_verifications Email баталгаажуулах OTP кодууд uid (doc ID), code, email, expiresAt (10 минут), createdAt

CI/CD автоматжуулалт

GitHub Actions хоолой (backend/ хавтаст push хийхэд автоматаар):

1. ✅ Docker дүрс build хийнэ
2. ✅ AWS ECR-д push хийнэ (ap-northeast-1, Токио)
3. ✅ EC2 дээр шинэ контейнер татаж ажиллуулна
4. ✅ Нийт байршуулалт: 3–4 минут
